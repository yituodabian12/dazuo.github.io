HTML   p标签

HTML 段落是通过 p 标签进行定义的。

浏览器会自动地在段落的前后添加空行。

HTML   br标签

br 可插入一个简单的换行符。

请注意，br 标签只是简单地开始新的一行，而当浏览器遇到 p 标签时，
通常会在相邻的段落之间插入一些垂直的间距。

HTML   h1标签

标题是通过 h1 - h6 等标签进行定义的。
h1 定义最大的标题。h6 定义最小的标题。

浏览器会自动地在标题的前后添加空行。

HTML   table标签

每个表格均有若干行,由 tr 标签定义，每行被分割为若干单元格,由 td 标签定义

数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等

HTML   ul & ol标签

无序列表是一个项目的列表，此列项目使用粗体圆点,典型的小黑圆圈进行标记。
无序列表始于 ul 标签。每个列表项始于 li。

同样，有序列表也是一列项目，列表项目使用数字进行标记。
有序列表始于 ol 标签。每个列表项始于 li 标签。

HTML   a标签

超链接可以是一个字，一个词，或者一组词，也可以是一幅图像，
您可以点击这些内容来跳转到新的文档或者当前文档中的某个部分。

    有两种使用a标签的方式：
  • 通过使用 href 属性 - 创建指向另一个文档的链接
  • 通过使用 name 属性 - 创建文档内的书签

CSS   animation

内含六个属性：
animation-name：代表要使用的动画名
animation-duration：动画播放一边所需时间
animation-timing-function：动画的播放速度函数
animation-delay：延时多久后开始播放
animation-iteration-count：动画播放次数
animation-direction：动画正放还是倒放，或穿插播放

    用keyframes来制作动画，内部可以用两种方法来定义中间状态
  • from to来定义动画初始状态和最终状态,但是只有两个中间状态
  • 用百分比来定义动画中间状态，可以有多个状态

CSS   background

内含六个属性
background-color：背景颜色
background-position：背景位置
background-size：背景大小
background-repeat：是否重复平铺
background-attachment：背景相对于屏幕位置是否固定
background-image：背景图片，用src加图片路径

当背景颜色与背景图片同时设置时，背景颜色会被覆盖
可以同时设置多个背景图片，定义时用逗号分隔不同图片的属性

CSS   border

内含四个属性
border-color：边框颜色
border-style：边框样式
border-width：边框宽度
border-radius：边框弧度

利用边框弧度可以做圆，椭圆等，而利用小区块与大边框可以做三角形与梯形

CSS   font

内含四个属性
font-size：字体大小
color：字体颜色
line-height：行与行之间的高度
font-style：字体样式

当背景颜色与背景图片同时设置时，背景颜色会被覆盖
可以同时设置多个背景图片，定义时用逗号分隔不同图片的属性

CSS   transform

内含四个属性
translate：左右上下平移
rotate：旋转
scale：缩放

可以通过旋转与平移边框来组成一些倾斜图形

JS   function

getElementById：通过ID获取元素
getElementsByClassName：通过类名获取元素
用一个var类型的变量储存获取到的元素
改变获取到元素的属性：变量名.style.CSS属性

JS   onclick

以下两种方法可以通过onclick定义点击事件
在HTML标签里加上onclick="要执行的代码"
元素.onclick=点击事件函数
